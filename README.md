## Prerequisites
+ Python: 3.7/3.8/3.9/3.10 (64-bit)
+ OS: Windows, OS X, and Linux (64-bit)
## Install Taichi
#### Taichi is available as a PyPI package:
`pip install taichi`
## Run Case
`python ./taichi_samples/**/main.py`
